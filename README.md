# AdPages Local SEO Audit Kit for Notion

Notion template pack for agencies, freelancers, and local businesses that need a lightweight place to plan campaign URLs, track landing-page QA, and follow up on leads.

This is a publishable template opportunity for Notion template marketplaces, Gumroad/Ko-fi freebies, GitHub, and support docs. It is useful as a free artifact by itself and can later point to paid exports, branded reports, or done-for-you audits.

## Included

- `templates/local-seo-audit-dashboard.md` - copyable Notion page structure.
- `data/audit-checklist.csv` - importable landing-page QA checklist.
- `data/campaign-tracker.csv` - importable UTM/campaign URL tracker.
- `data/lead-follow-up.csv` - importable lead follow-up tracker.
- `PRIVACY.md` - privacy and data-handling notes.

## Local Checks

From the repository root:

```bash
npm --prefix templates/notion/adpages-local-seo-audit-kit run check
npm --prefix templates/notion/adpages-local-seo-audit-kit run smoke
```

The checks validate package JSON, required files, CSV headers, template placeholders, and make sure the template contains no scripts, remote embeds, secrets, or hidden tracking instructions.

## Notion Setup

1. Create a new Notion page called `AdPages Local SEO Audit Kit`.
2. Copy the dashboard markdown from `templates/local-seo-audit-dashboard.md`.
3. Import each CSV as its own database:
   - `Landing Page QA`
   - `Campaign URLs`
   - `Lead Follow-Up`
4. Link the three databases into the dashboard page.
5. Replace example domains, services, and owners before sharing with a client.
