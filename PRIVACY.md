# Privacy Notes

This Notion template pack is static. It does not include scripts, automations, API calls, analytics pixels, hidden backlinks, or remote embeds.

Any personal data entered into an imported Notion database is controlled by the workspace owner and Notion's platform terms. Before sharing a workspace with clients or staff, review database permissions, retention expectations, and whether lead data should be stored there at all.

Do not publish real lead details, phone numbers, email addresses, client notes, or campaign performance data in a public duplicated template.
